## 💫 About Me

I specialize in constructing robust backend systems and crafting seamless, highly responsive user interfaces. My design philosophy leans heavily toward minimalist, ultra-clean layouts, sophisticated dark aesthetics, and fluid user interactions.

- 📱 **Mobile Development**: Crafting cross-platform apps using React Native & Expo.
- 💻 **Web Ecosystems**: Engineering scalable architectures with Laravel, PHP, and React.
